Community Modules

This repository provides a collection of reusable Kubernetes module templates.

Aggregated Module Files

  • all-modules.json — All modules as a single JSON array
  • all-modules.yaml — All modules as YAML documents

Install All Modules at Once

To install all community modules in your cluster with a single command, run:
kubectl apply -f https://kyma-project.github.io/community-modules/all-modules.yaml

For more details, see the README.md of the project.
