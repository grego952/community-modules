If the page is not loading, please download the latest version and save it in some directory. You can do this also from command line: 
curl https://kyma-project.github.io/community-modules/index.html -o index.html
Then execute the following command to start the local server: 
kubectl proxy -w='.'
and open Web UI with this link: http://127.0.0.1:8001/static/
